1、使用局部变量ListNode head，返回head.next（head会自动销毁）的方式可以避免需要在循环中记录头指针；
2、对于奇数数组，a[n/2]=a[(n-1)/2],对于偶数数组，a[n/2]表示中间偏右，a[(n-1)/2]表示中间偏左；
3、首尾接近法：
	s = 0;
	e = len - 1;
	while (s < e)
{
	if ... s++;
	else if ... e--;
}
4、二分法：
	s = 0
	e = len - 1
	while (s <= e)
{
	i = （s + e）/ 2
	if ...
		s = i + 1
	else if ...
		e = i - 1
}
